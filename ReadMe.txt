This is a simple nodejs Web API that demostrate web scrapping.
I made python script i.e webscrapper.py which is used for web scraping of NEPSE website.
For now it only fetch NEPSE Index value.

Requirement
1. Node.js
2. python 3
3. vscode

How to run this project?
1. Install Node.js on your computer
2. Open the project in vscode
3. Open terminal in vscode (ctr + `)
4. Install typescipt using terminal (npm install -g typescript)
5. Install node modules (npm install)
6. Run the project (F5)
7. Call API (localhost:3000/api/nespseindex/) from browser or postman
8. You Should get current NEPSE Index
